## 🏗️ Architecture Overview

This project follows a **Monorepo Architecture** powered by:

- ⚡ Turborepo / PNPM Workspaces
- 🧩 Modular Feature-Driven Design (FDD)
- 🧱 Atomic Design System (UI)
- 🧠 Domain-driven backend structure
- ☁️ Cloud-ready infrastructure (Docker + K8s + Terraform)

### Frontend
- ⚛️ Next.js 14+ (App Router)
- 🎨 Tailwind CSS
- 🧠 TypeScript
- 📊 D3.js / Chart.js

### Backend
- 🟢 Node.js / Express (Modular API)
